### Lua REPL

Run `lua` from Xyron to drop into an interactive Lua session with full access to the `xyron.*` API. Useful for testing config snippets, exploring the API, or quick scripting without editing `config.lua`.

### CWD Reporting Fix

Fixed an issue where Attyx didn't know Xyron's working directory until the first `cd`. Xyron now reports its initial CWD on startup, so terminal features that depend on the current directory (like tab titles and CWD display) work immediately.
