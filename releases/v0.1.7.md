### Fuzzy Completions

Completions now use fuzzy matching — type any part of a command, file, or flag name and matching candidates surface automatically. The completion buffer has been increased to handle larger candidate sets.

### Tab and Ctrl+Space

Tab and Ctrl+Space now have distinct roles:

- **Ctrl+Space** — opens the completion overlay to browse candidates
- **Tab** — completes immediately: accepts the selected candidate if the overlay is open, or triggers completion and auto-accepts if there's a single match

When the typed text exactly matches the top candidate, the overlay dismisses automatically — pressing Enter executes the command directly without inserting a trailing space.

### Block UI Removed

The block UI rendering layer has been removed in favor of simpler, more reliable output handling.
