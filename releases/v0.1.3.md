### Smart Completions

Tab completion now has context-aware providers for common tools. Git subcommands, branches, and remotes complete automatically. So do flags and arguments for `docker`, `brew`, `cargo`, `npm`, `bun`, `zig`, `ssh`, `systemctl`, and more.

Custom completions from Lua are also supported — use `xyron.complete()` in your config to register completions for any command.

### Completion Fixes

- Fixed picker selection getting out of sync with visible items
- Fixed overlay positioning when completions appear near the bottom of the terminal
- Reduced flickering during rapid tab cycling

### Stability

- Fixed crash when `$PATH` contains entries the shell can't read
- History navigation no longer panics on very long input lines
- Headless mode handles malformed protocol frames gracefully instead of crashing
